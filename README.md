# Stock Analysis Dashboard

An interactive stock analysis platform built using **Python and Streamlit** that allows users to search global stocks, analyze price movements, and simulate investment strategies.

---

## Features

- Search stocks using company name or ticker
- Interactive price charts (Line & Candlestick)
- Technical indicators such as Moving Averages and Volume
- Investment simulator for Lump Sum and SIP strategies
- Performance metrics including returns, 52-week high/low, and trading volume
- Fundamental metrics like Market Cap, P/E Ratio, EPS, and Beta

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- yfinance
- Requests

---

## How to Run the Project

Clone the repository:

```bash
git clone https://github.com/vaasu-kohli/stock-analysis-dashboard.git
cd stock-analysis-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run stock_project_final.py
```

The dashboard will open in your browser.
